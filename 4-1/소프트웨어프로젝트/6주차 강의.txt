* 인공지능
-인간 수준의 지능을 갖는 컴퓨터 실현을 위한 SW기술
-인간의 학습능력, 추론능력, 지각능력, 자연언어의 이해 능력 등을 컴퓨터 프로그램으로 실현한 기술
-컴퓨터가 인간의 지능적인 활동을 모방할 수 있도록 하는 것 

* 인공지능의 영역 : 언어지능, 시각지능, 학습지능, 뇌인지컴퓨팅 등
-자연언어 처리 분야(자연언어의 이해가 필요), 
 전문가시스템 분야(특정 영역에 있어 그 영역의 전문가만 성능을 내는 인공지능 시스템 ex)의료 분야),
 영상 및 음성 인식 분야(이미지를 보고 알아내고 이미지를 유사한 것끼리 모아놓는 것, 유사한 것끼리 분류해놓는 것), 
 이론증명(어떠한 것을 증명해내는것), 
 신경망(Newral network)

* AI : 주변환경을 인식하고, 목표를 성공적으로 성취할 기회, 즉 확률을 최대화하기위해 어떤 행위를 취하는데 이런 똑똑한 Agent
Symbolic AI : 정통적인 AI, 물리적 symbol system 가설을 기반,  우리가 아는 걸 기호로 표현하고 컴퓨터는 이러한 기호를 이용해서 조작 => 기호를 처음에 우리가 가설을 했던 환경으로 되돌려버림
                   ex) 사람은 눈이 2개다. 이런 지식이 주어질 때 이것은 사람인가 추론하는 것
	     현실세계 우리가 아는 것들을 어떻게 지식으로 표현해서 처리할까가 중요

CI : 모든 데이터들을 기반으로 그 데이터들을 통계적으로 처리
      통계적인 처리를 기반으로 분류, 인식, 추론을 함
=> Newral network(신경망), 
     Fuzzy logic(속하는 지수를 0과 1사이의 지수로 나타내는 것, 이러한 확률로 표현하는 것),
     Evolutionary computation(내가 어떤 상황에서 어떤 결론을 냈을 때 똑같은 결론이 나지 않도록 진화론적 계산을 수행하는 것),
     Learning theory(배우는 것),
     Probabilistic methods(확률 기반 학습)

학습 수준 : 1. 기억하는 것, 2. 이해하는 것, 3. 적용하는 것, 4. 분석하는 것, 5. 종합하는 것, 6. 평가하는 것

AI : Machine Learning, Expert System, Computational Intelligence, Distributed Artificial Intelligence, Intelligent Interfaces

인공지능 : 특정 분야에서 인간의 능력을 뛰어넘는 기계 
ex)딥 블루가 체스분야에서 세계랭킹 1위를 이김, 왓슨이 퀴즈 분야에서 인간 상대 승리, 알파고(Go : 바둑)가 바둑 분야에서 이세돌을 이김 => 경우의 수는 무한대이나 패턴으로 이김

인공지능 비서 : 시리, 빅스비, 나우, 알렉사, 코타나, M 

제프리 힌튼 : 딥러닝의 아버지(꽃을 분류하는 것을 수행), 
                   DNN 리서치 => 구글이 인수
