4차 산업혁명

산업혁명 : 아놀드 토인비가 처음 사용,  사회, 경제 구조의 변혁, 생산성 혁명
1차 : 증기기관(기계화), 단순 육체노동 대체 -> 소유 => 제국주의
2차 : 전기(공업화), 주요 육체노동 대체 -> 소유 => 자본주의
3차 : 디지털(정보화), 단순 지식노동 대체 -> 개방 => 협업적 공유사회
4차 : 사이버물리시스템(초연결, 초지능, 초실감), 고급 지식노동 대체 -> 공유 => 초연결, 초지능적 사회

4차 산업혁명 : 차세대 정보통신기술, 로봇 기술, 생명 기술 등의 융합
->초연결(IoT, 클라우드, 유비쿼터스), 초지능(인공지능, 빅데이터), 초실감 기술(오감센싱, 홀로그램, AR, VR)
=>디지털 유기체 생태계(플랫폼)로 진화

기반 학문 분야 : IT, 전자공학, 로봇공학, 생명공학, 나노공학
기반 기술 : 인공지능, 사물인터넷, 빅데이터 등 + 클라우드, 3D 테크놀로지

ICT 파도
제1차 : 디지털화
제2차 : 사이버공간 탄생(인터넷, www)
제3차 : 물리적 공간과 사이버공간 연결 확장
제4차 : 초연결, 초지능, 초실감 빅뱅 기반 초증강현실

가트너 주기
기술 촉발 -> 부풀려진 기대의 정점 -> 환멸 -> 계몽 -> 생산성 안정
